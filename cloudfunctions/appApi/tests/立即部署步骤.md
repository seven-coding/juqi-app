# 立即部署步骤

## 重要提示

由于当前环境没有配置腾讯云MCP服务器，**推荐使用微信开发者工具进行部署**，这是最简单可靠的方式。

---

## 部署步骤（使用微信开发者工具）

### 第一步：部署核心层云函数（必须先部署）

1. **打开微信开发者工具**
2. **打开项目**: `JUQI-小程序`
3. **进入云开发**:
   - 点击工具栏"云开发"按钮
   - 或：工具 → 云开发 → 云开发控制台
4. **部署云函数**（按顺序）:
   - **getUserList**: 右键 → "上传并部署：云端安装依赖" ✅
   - **getDynsListV2**: 右键 → "上传并部署：云端安装依赖" ✅
   - **publishDyn**: 右键 → "上传并部署：云端安装依赖" ✅

### 第二步：部署 appApi 云函数

1. **切换到项目**: `JUQI-APP`
2. **进入云开发控制台**
3. **部署 appApi**:
   - 找到 `appApi` 云函数
   - 右键 → "上传并部署：云端安装依赖" ✅

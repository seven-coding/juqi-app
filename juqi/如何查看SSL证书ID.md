# 如何查看 SSL 证书 ID

## 方法一：在云开发控制台查看（推荐）

### 步骤：

1. **登录云开发控制台**
   - 访问：https://tcb.cloud.tencent.com/dev?envId=prod-juqi-7glu2m8qfa31e13f#/static-hosting

2. **进入静态网站托管**
   - 点击左侧菜单"静态网站托管"
   - 或直接访问上面的链接

3. **查看域名配置**
   - 在"自定义域名"或"域名管理"区域
   - 如果已经配置过域名，可以看到域名列表
   - 点击域名，可以看到详细信息，包括证书 ID

4. **如果没有配置过域名**
   - 点击"添加域名"
   - 输入域名后，系统会提示选择证书
   - 如果已有证书，会显示证书列表和证书 ID
   - 如果没有证书，可以选择"自动申请免费证书"

## 方法二：在腾讯云 SSL 证书控制台查看

### 步骤：

1. **登录腾讯云控制台**
   - 访问：https://console.cloud.tencent.com/ssl

2. **查看证书列表**
   - 在"我的证书"页面
   - 可以看到所有已申请的证书
   - 证书 ID 通常显示在证书列表中，格式类似：`cert-xxxxxxxx`

3. **证书详情**
   - 点击证书名称，进入详情页
   - 可以看到完整的证书 ID

## 方法三：在域名配置时查看

### 步骤：

1. **在云开发控制台添加域名时**
   - 进入静态网站托管
   - 点击"添加域名"
   - 输入域名：`juqi.life`
   - 在"SSL 证书"选择中：
     - 如果选择"使用已有证书"，会显示证书列表，每个证书旁边会显示证书 ID
     - 如果选择"自动申请免费证书"，系统会自动申请，申请成功后也会显示证书 ID

## 📝 证书 ID 格式

证书 ID 通常格式为：
- `cert-xxxxxxxx`（腾讯云格式）
- 或一串数字和字母组合

## 🔍 如果没有证书怎么办？

### 选项 1：使用云开发自动申请免费证书（推荐）

在添加域名时，选择"自动申请免费证书"：
- ✅ 免费
- ✅ 自动续期
- ✅ 配置简单
- ⏱️ 需要几分钟申请时间

### 选项 2：使用已有证书

如果你在其他地方（如腾讯云 SSL 证书控制台）已经有证书：
1. 在云开发控制台添加域名时
2. 选择"使用已有证书"
3. 从列表中选择证书
4. 记录证书 ID

### 选项 3：上传新证书

如果你有证书文件（.crt 和 .key）：
1. 先在腾讯云 SSL 证书控制台上传证书
2. 获取证书 ID
3. 在云开发控制台配置域名时选择该证书

## 🚀 快速操作建议

**最简单的方式**：
1. 直接去云开发控制台添加域名
2. 选择"自动申请免费证书"
3. 等待证书申请完成（通常几分钟）
4. 配置 DNS 解析
5. 完成后告诉我，我帮你更新代码

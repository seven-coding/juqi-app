# 最简单的解决方案：通过腾讯云控制台修改 Content-Type

## 🎯 问题

COSCLI 下载失败，文件只有 9 字节。最简单的方法是**直接通过腾讯云 COS 控制台修改**，无需安装任何工具。

## ✅ 解决方案：使用腾讯云控制台

### 步骤 1：访问 COS 控制台

直接访问以下链接（会自动跳转到登录页面）：

**方法一：直接链接**
```
https://console.cloud.tencent.com/cos/bucket?bucket=a0d1-static-prod-juqi-7glu2m8qfa31e13f-1314478640&region=ap-shanghai
```

**方法二：手动导航**
1. 访问：https://console.cloud.tencent.com/cos
2. 找到存储桶：`a0d1-static-prod-juqi-7glu2m8qfa31e13f-1314478640`
3. 选择地域：`ap-shanghai`（上海）

### 步骤 2：找到文件

1. 进入存储桶后，点击左侧的"文件列表"
2. 导航到 `.well-known` 文件夹
3. 找到 `apple-app-site-association` 文件

### 步骤 3：修改元数据

**方法 A：通过文件详情修改**

1. 点击 `apple-app-site-association` 文件右侧的 **"详情"** 按钮
2. 在详情页面找到 **"元数据"** 或 **"HTTP 头"** 部分
3. 找到 `Content-Type` 字段
4. 将值从 `application/octet-stream` 改为 `application/json`
5. 点击 **"保存"**

**方法 B：通过文件操作菜单修改**

1. 点击 `apple-app-site-association` 文件右侧的 **"更多"** 或 **"..."** 菜单
2. 选择 **"修改元数据"** 或 **"编辑属性"**
3. 修改 `Content-Type` 为 `application/json`
4. 保存

**方法 C：如果找不到修改选项**

如果控制台界面没有直接的元数据修改选项，可以：

1. **删除旧文件**
   - 选中文件，点击"删除"

2. **重新上传文件**
   - 点击"上传"
   - 选择本地文件：`JUQI-APP/juqi/apple-app-site-association`
   - 在上传对话框中，找到"高级设置"或"元数据"
   - 设置 `Content-Type` 为 `application/json`
   - 上传到路径：`.well-known/apple-app-site-association`

### 步骤 4：验证修改

修改后，等待 1-2 分钟，然后在终端执行：

```bash
curl -I https://prod-juqi-7glu2m8qfa31e13f-1314478640.tcloudbaseapp.com/.well-known/apple-app-site-association | grep -i content-type
```

应该看到：`content-type: application/json`

或者在浏览器中访问：
```
https://prod-juqi-7glu2m8qfa31e13f-1314478640.tcloudbaseapp.com/.well-known/apple-app-site-association
```

应该直接显示 JSON 内容，而不是下载。
